# Jeff Sun Trading Wiki

A structured knowledge base built from Jeff Sun's (@jfsrev) trading system, sourced from his tweets, substacks, and community compilations.

## Structure

- `wiki/` — Synthesized knowledge articles (~40 pages)
- `sources/tweets/` — Individual tweet content (~200 files)
- `sources/substacks/` — Full substack articles
- `sources/notion/` — Snapshots of original Notion pages
- `examples/` — Detailed trade breakdowns (KC, COIN, RGTI, etc.)
- `_meta/` — Map of Content, Glossary, Tags (Dataview-powered)
- `slides/` — Marp presentation decks

## Getting Started

1. Open this folder as a vault in Obsidian
2. Install the Dataview community plugin (config is pre-set)
3. Start at `_meta/MOC.md` — the Map of Content

## Source Attribution

All content is attributed to Jeff Sun (@jfsrev) and referenced authors. Original tweet URLs and Notion page IDs are preserved in frontmatter.
