Fluxus Trading Risk Manager — Description & Usage Guide
=======================================================

WHAT IT DOES
------------
A TradingView overlay indicator that calculates position size and manages
risk for any stock or ETF. Given your account size, risk tolerance, entry
price, and a stop level, it tells you how many shares to buy and draws
stop/target lines on the chart. It also supports graduated (N-stop) exits
and automatic short-side mirroring.

Two calculation modes:
  - "Shares to Buy" — you set entry + stop, it returns share count.
  - "Stop Loss"     — you set shares owned, it returns stop price.

Trade direction toggle (Long/Short) — when set to Short, the stop
validation and major level auto-mirror flip automatically so stop-above-
entry is treated as correct.


FEATURES
--------
- Risk-based position sizing (0.25% / 0.50% / 0.75% / 1.00% or custom %)
- Position size capping (max % of portfolio)
- Fractional share support
- 15 stop-level methods: LOD, HOD, current/previous bar, previous day,
  swing pivot, ATR-offset, MA, MA structure high/low
- Auto-mirror stops for short positions (see below)
- N-Stop graduated exits (2-stop or 3-stop)
- Configurable R-multiple profit targets (default 2R / 3R / 5R)
- Configurable multiplier columns (full-size / half-size)
- On-chart info table with customizable position, size, and colors


SETTINGS GROUPS
---------------

1. Account & Risk
   - Account Size: total portfolio value in dollars.
   - Equity Risk: preset (0.25-1.00%) or toggle "Custom Equity Risk" for
     any percentage. This is the max dollar amount you lose if stopped out.
   - Max Position Size: optional cap as % of portfolio.
   - Calculate mode: "Shares to Buy" or "Stop Loss".
   - Direction: Long or Short — controls stop validation and auto-mirror.
   - Fractional Shares: round to 2 decimals instead of flooring.

2. Price Levels
   - Entry Price: manual dollar value, or toggle "Use Candle Levels" to
     auto-pick from Current Price / High / Low / Previous High / Previous Low.
   - Stop Price: manual dollar value, or toggle "Use Major Level" to pick
     from one of 15 methods (see list below).
   - Buffer: add a % or $ cushion beyond the raw stop level.
   - Shares Owned: used only in "Stop Loss" calc mode.

3. Major Level Settings
   - Pivot Length: lookback bars for swing high/low detection.
   - ATR Length + Multiplier: for ATR-based stop methods.
   - Moving Average: length + type (EMA or SMA) for MA-based stops.

4. N-Stop Risk Management
   - Enable N-Stop Levels: turn graduated exits on/off.
   - Number of Stops: 2 or 3.
   - Show N-Stop Table Rows: add stop details to the info table.
   - Show R-Multiple Targets: draw profit target lines on chart.
   - R-Targets: three configurable multipliers (default 2R / 3R / 5R).
   - Line colors for stops and targets.

5. Display Settings
   - Show Stop Loss on Chart: draw the main stop line.
   - Include Short Info: add mirrored short calculations to the table.
   - Table position, text size, text color, background color.

6. Multiplier Display
   - Full / Half multipliers applied to share count — useful for scaling
     into positions (e.g., 2.1x for a full-conviction add, 0.56x for a
     starter position).


STOP LEVEL METHODS (Major Levels dropdown)
------------------------------------------
Long side (selected directly):       Short side (auto-mirrored):
  Low of Day                           -> High of Day
  Current Bar Low                      -> Current Bar High
  Previous Bar Low                     -> Previous Bar High
  Previous Day Low                     -> Previous Day High
  Swing Low                            -> Swing High
  % ATR from LOD                       -> % ATR from HOD
  % ATR from Entry                     -> % ATR from Entry (above)
  % ATR from Previous Bar Low          -> % ATR from Previous Bar High
  MA (close-based)                     -> MA (close-based)
  MA Structure Low                     -> MA Structure High
  High of Day                          -> Low of Day
  Previous High                        -> Previous Day Low
  Swing High                           -> Swing Low
  % ATR from HOD                       -> % ATR from LOD
  MA Structure High                    -> MA Structure Low


AUTO-MIRROR CONCEPT
-------------------
You pick ONE stop method from the dropdown. The indicator automatically
computes the opposite level for short positions. If you select "Low of Day"
as your stop, a long trade uses LOD as the stop; if you toggle on "Include
Short Info", the short trade automatically uses High of Day. Swing Low
mirrors to Swing High, ATR below LOD mirrors to ATR above HOD, and so on.

This means you never need separate long/short configurations — the
indicator handles both directions from a single setting.


N-STOP GRADUATED EXITS
-----------------------
Instead of exiting the full position at one stop, N-Stop splits it into
equal tranches and assigns graduated stop levels.

Math:
  R = |entry - full_stop|           (total risk distance)
  Stop_k = entry - (k/N) * R       (for longs; reversed for shorts)
  Qty per tranche = floor(shares / N), last tranche gets the remainder

2-Stop example (entry $100, stop $96, R = $4, 100 shares):
  SL1 = $98 (1/2 R) x 50 shares    -> loss = $100
  SL2 = $96 (2/2 R) x 50 shares    -> loss = $200
  Total graduated loss = $300 vs $400 if all 100 hit the full stop.

3-Stop example (same setup, 99 shares):
  SL1 = $98.67 (1/3 R) x 33 shares -> loss =  $43.89
  SL2 = $97.33 (2/3 R) x 33 shares -> loss =  $88.11
  SL3 = $96.00 (3/3 R) x 33 shares -> loss = $132.00
  Total graduated loss = $264 vs $396 full stop.

The info table shows the graduated loss vs full-stop loss so you can see
the savings. R-multiple targets are drawn on the profit side as fixed-
length dashed lines (20 bars) for take-profit reference. The R-multiples
are configurable (default 2R, 3R, 5R).


QUICK START
-----------
1. Set Account Size and Equity Risk to match your portfolio.
2. Pick an entry method (candle levels or manual price).
3. Pick a stop method (major level or manual price), add a buffer if needed.
4. Enable N-Stop if you want graduated exits.
5. Read the shares count and position size from the table. Done.
